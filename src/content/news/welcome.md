# 欢迎使用 MosuoAI 主题

这是 MosuoAI 主题的示例文章。您可以删除此文章并开始撰写您自己的内容。

## 主题特性

- 🎨 精美的暗色/亮色主题切换
- 📱 完全响应式设计
- 🔍 内置搜索功能
- 📝 Markdown 支持
- 🏷️ 标签分类系统
- 📊 多种内容类型支持（资讯、教程、评测、开源、资源）

## 快速开始

1. 克隆此主题
2. 安装依赖：`npm install`
3. 启动开发服务器：`npm run dev`
4. 开始撰写您的文章！

祝您使用愉快！
